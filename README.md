# Machine Learning Algorithms – Supervised & Unsupervised

This repository contains implementations of **Supervised and Unsupervised Machine Learning algorithms** using **Python and Scikit-learn**.
The goal of this project is to understand the **core concepts, implementation, and practical usage of common ML algorithms**.

These implementations demonstrate how different algorithms learn patterns from data to make predictions or discover hidden structures.

---

# Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib (for visualization)

---

# Supervised Learning Algorithms

Supervised learning algorithms are trained using **labeled datasets**, where the model learns the relationship between input features and the target output.

Algorithms included in this repository:

* Linear Regression
* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)

Applications of supervised learning:

* Price prediction
* Disease prediction
* Spam detection
* Classification problems

---

# Unsupervised Learning Algorithms

Unsupervised learning algorithms analyze **unlabeled data** to find hidden patterns, clusters, or structures.

Algorithms included in this repository:

* K-Means Clustering
* Hierarchical Clustering
* Principal Component Analysis (PCA)
* DBSCAN (optional if you add)

Applications of unsupervised learning:

* Customer segmentation
* Market analysis
* Pattern discovery
* Dimensionality reduction
